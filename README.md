# Swiggy Clone

A frontend clone of the Swiggy homepage built using HTML and CSS.

## Technologies Used
- HTML5
- CSS3

## Features
- Navigation Bar
- Hero Section
- Food Cards
- Footer

## Purpose
This project was created to practice HTML and CSS and understand how real-world websites are structured.

## Note
This is a learning project inspired by the Swiggy website and is not affiliated with Swiggy.
